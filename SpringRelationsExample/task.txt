Дополните проект SpringRelationsExample.zip (см. дискорд),
с которым мы работали на лекции следующими сущностями:
UserVerificationData – сущность хранит данные, используемые
для верификации пользователя (фотография, запись голоса,
биометрические данные, статус верификации). У одного пользователя есть один набор
верификационных данных.
Статус пользователя UserStatus (поля isVip и grade – SILVER, GOLD, PLATINUM, NONE).
У пользователя может быть только один текущий статус. Один и тот же статус может быть
у нескольких пользователей.
ExpertGroup – экспертная группа, к которой относится пользователь на основании
оставленных комментариях о товарах (имя группы и пользователи, относящиеся к этой группе).
Например, название «Эксперт по холодильникам», «Специалист по огурцам».
Пользователь может состоять во многих группах. В одной группе может быть много пользователей.
